# 赵霄 · 数字简历

深色科技风、**自包含、零依赖**的单页数字简历,供 HR / 招聘方在线浏览、打印或存 PDF。

> 在线访问:<https://zhaoxiao5174-beep.github.io/resume/>

## 仓库内容
- `index.html` — 简历主体(单文件,内联 CSS/JS,响应式,自带打印样式)
- `resume.pdf` — 「下载 PDF」按钮指向的**最新**简历 PDF
- `赵霄-简历-算力中心基建.docx` — 可编辑源文件(算力中心基建岗 · 最新版)

## 本地预览
双击 `index.html`;或 `python -m http.server 8000` 后访问 `http://localhost:8000`。

## 部署
push 到 `main` 后 GitHub Pages 自动更新(首次:Settings → Pages → Source = Deploy from branch → main / root)。

## 更新简历
替换 `resume.pdf` 与 `赵霄-简历-算力中心基建.docx`(由 `resume-web/` 的 `resume-source.*` 同步生成)→ `git commit` → `git push`。

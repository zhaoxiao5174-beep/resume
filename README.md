# 赵霄 · 数字简历

深色科技风、**自包含、零依赖**的单页数字简历,供 HR / 招聘方在线浏览、打印或存 PDF。

> 在线访问:<https://zhaoxiao5174-beep.github.io/resume/>

## 仓库内容
- `index.html` — 简历主体(单文件,内联 CSS/JS,响应式,自带打印样式)
- `赵霄-简历-DeepSeek-IDC设计规划.pdf` — 「下载 PDF」按钮指向的版本

## 本地预览
双击 `index.html` 用浏览器打开;或在该目录执行 `python -m http.server 8000` 后访问 `http://localhost:8000`。

## 部署
push 到 `main` 后,GitHub Pages 自动更新(首次需在 Settings → Pages 开启:Source = Deploy from branch → main / root)。

## 修改简历
直接编辑 `index.html` → `git commit` → `git push`。无需构建步骤。
